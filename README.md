# bookthatapp-sdk-example

You can import `bookthatapp-sdk` by `script tag` or `import` or `require`.

### import

1. `yarn importExample` 
2. http://localhost:9001/test

### require

1. `yarn importExample` or `yarn requireExample` 
2. http://localhost:9001/test


### script tag

1. open `src/script.html`
